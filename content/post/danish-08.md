## Grammar Summary (Beginner Level)

### 1. Passive Voice with "-s" Ending
Danish uses **-s** to create passive voice:
- **tilbydes** = is offered (from tilbyde)
- **søges** = is wanted (from søge)
- **ses** = see each other (reflexive)

**Examples**:
- Professionel rengøring tilbydes = Professional cleaning is offered
- Rengøringsassistent søges = Cleaning assistant is wanted
- Vi ses! = See you! (literally: we see each other)

### 2. Compound Words (Sammensatte ord)
Danish creates complex words by combining simpler ones:
- **støvsugning** = støv (dust) + suge (suck) + ning (noun ending) = vacuuming
- **vinduespolering** = vindue (window) + polering (polishing) = window polishing
- **kulturhus** = kultur (culture) + hus (house) = cultural center
- **billedbehandling** = billede (image) + behandling (treatment) = image processing
- **lektiehjælp** = lektie (homework) + hjælp (help) = homework help
- **skolearbejdet** = skole (school) + arbejdet (the work) = the school work
- **arbejdsstillinger** = arbejds (work) + stillinger (positions) = working positions
- **dyrearter** = dyre (animal) + arter (species) = animal species
- **landområder** = land (country) + områder (areas) = rural areas
- **kæledyr** = kæle (cuddle) + dyr (animal) = pets
- **varmrøget** = varm (warm) + røget (smoked) = hot-smoked
- **laksefilet** = laks (salmon) + filet (fillet) = salmon fillet
- **hjemmelavede** = hjemme (home) + lavet (made) + e (plural) = homemade
- **rejesalat** = reje (shrimp) + salat (salad) = shrimp salad

### 3. "Op-" Prefix Words
**Op-** prefix often indicates creation or upward action:
- **oprettelse** = creation (from oprette - to create)
- **opstart** = startup (from starte - to start)
- **opleve** = experience (from leve - to live)

### 4. Superlative with "-st"
Add **-st** to adjectives for superlative:
- **hurtigst** muligt = fastest possible (as soon as possible)
- **hurtig** → **hurtigst** = fast → fastest

### 5. Adjective Agreement
Adjectives change based on definiteness and number:
- Indefinite singular: **en stabil** person = a stable person
- Definite/plural: **de frivillige** = the volunteers
- Plural: **friske bude** = fresh couriers

### 6. Definite Forms ("the")
Add endings to make nouns definite:
- **-en**: alder → **alderen** (the age)
- **-et**: arbejdet → **arbejdet** (the work), havet → **havet** (the sea)
- **-ne**: fagene → **fagene** (the subjects)

### 7. "Vild med" - Crazy About
Idiomatic expression:
- **Vild med** dyr? = Crazy about animals?
- **Vild med** musik = Crazy about music

### 8. Impersonal "Man"
**Man** is used as generic "one/you":
- **som man kan** opleve = that one/you can experience
- **Man skal** = One must/You must

### 9. Present Tense Formation
Most verbs add **-r** or **-er**:
- **søge** → **søger** = seek → seeks
- **mangle** → **mangler** = lack → lacks
- **tjene** → **tjener** = earn → earns

### 10. Idiomatic Phrases
Common Danish expressions:
- **Kig forbi** = drop by (literally: look past)
- **Bare kig forbi** = just drop by
- **Vi ses** = see you (literally: we see each other)
- **Det hele** = all of it
- **Lidt ekstra** = a little extra
- **Tæt på** = close to

### 11. "Hurtigst muligt" - As Soon As Possible
Common phrase structure:
- **hurtigst muligt** = as fast as possible
- Pattern: superlative + muligt (possible)

### 12. Prepositions
Common prepositions in context:
- **til** = to/for (Alt til kæledyr = Everything for pets)
- **i** = in (I alderen 7-15 = In the age 7-15)
- **af** = of (Rengøring af toiletter = Cleaning of toilets)
- **ved** = at/by (Ved døren = At the door)
- **fra** = from (Fra havet = From the sea)
- **på** = on (På legetøj = On toys)

### 13. Negation Patterns
**Ikke** is used for negation but can be part of compounds:
- **ikke-ryger** = non-smoker (compound word)

### 14. Plural Formation
Add **-e** or **-er** for plural:
- frivillig → **frivillige** = volunteer → volunteers
- område → **områder** = area → areas
- art → **arter** = species → species (plural)

### 15. "Alt" Constructions
**Alt** can mean "all" or "everything":
- **Alt til** kæledyr = Everything for pets
- **Det hele** = All of it
- **Alt godt** = All good

### 16. Professional/Occupational Titles
Common professional terms:
- **rengøringsassistent** = cleaning assistant
- **pensionist** = pensioner
- **studerende** = student
- **frivillige** = volunteers

### 17. Measurement Expressions
- **pr.** = per (pr. 100 gram = per 100 grams)
- **ca.** = cirka = approximately (ca. 350 gram)
- **stk.** = stykke = piece (1 stk.)

### 18. "Både ... og" - Both ... and
Used to connect options:
- **Både** by- **og** landområder = Both urban and rural areas
